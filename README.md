# Branche un_agent_unity

Nous avons décidé de passer à Unity. Cette branche contient la version du jeu Overcooked avec un seul agent, développée avec Unity.

Ci-dessous une vidéo démonstrative de cette version du jeu :

[Regarde la vidéo Solo Agent](https://youtu.be/dI9PQ1T7Whc)

## Résumé des branches principales du projet

- **main** : contient la dernière version du jeu, avec plusieurs agents (5 agents) et développée avec Unity.
- **automaticalReceipe** : contient le projet 1, le jeu Overcooked avec un seul agent développé avec Godot.
- **un_agent_unity** : contient la version du jeu Overcooked avec un seul agent développée avec Unity.
- **double-agents** : contient la version du jeu Overcooked avec deux agents développée avec Unity.
- **multi-agent-unity** : contient la version du jeu avec plusieurs agents (5 agents) développée avec Unity.
