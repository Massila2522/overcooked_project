Branche double-agents

Cette branche contient la version du jeu Overcooked développée avec Unity, intégrant deux agents travaillant simultanément dans la cuisine.
Elle représente une étape intermédiaire entre la version à un seul agent et la version finale multi-agents.

 Objectif de cette branche

L’objectif de double-agents est d’introduire la coopération entre deux agents afin de :

tester la coordination des tâches (collecte d’ingrédients, cuisson, service, etc.) ;

analyser l’impact de l’ajout d’un deuxième agent sur la performance globale ;

préparer la transition vers la version à 5 agents disponible dans la branche main.
